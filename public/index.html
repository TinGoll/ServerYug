API Справочник

Адрес запроса Описание
http://192.168.2.10:3131/orders GET Запрос: Возвращает массив заказов [orders], общее колличество заказов count и колличество страниц pages.
Параметры: _sort, _order, _page, _limit
http://192.168.2.10:3131/orders/ID(number) GET Запрос: Возвращает заказ order, шапку header, тело заказа body и планы plans.
Параметры: Нет
http://192.168.2.10:3131/orders/sample/ID(number) GET Запрос: Возвращает изображение образца, которое находиться в папке заказов "Образец".
Параметры: Нет
http://192.168.2.10:3131/api/journals/get-journals GET Запрос: Возвращает список доступных журналов, требуеться Token.
Параметры: Нет
http://192.168.2.10:3131/api/journals/ID(number) GET Запрос: Возвращает содержимое журнала по ID, тербуеться Token.
Параметры: Нет
http://192.168.2.10:3131/api/at-order/data GET Запрос: Возвращает список Barcodes.
Параметры: Нет
http://192.168.2.10:3131/api/journals/adopted/ID(number) GET Запрос: Возвращает список принятых заказов, кол-во всего и кол-во страниц.
Параметры:_limit=300 - кол-во строк, _page=1 - страница, _d1=01.09.2021 - дата начала фильтрации, _d2=01.09.2021 - дата конец фильтрации, _filter=москва - умный фильтр
http://192.168.2.10:3131/api/at-order/salary-transactions/ID(number) GET Запрос: Возвращает список транзакций по зарплате журнала. ID - id журнала.
Параметры: Нет
http://192.168.2.10:3131/api/at-order/salary-report/:idtransaction(number) GET Запрос: Возвращает данные по зарплате по номеру транзакции
Параметры: Нет
http://192.168.2.10:3131/api/at-order/add POST Запрос: Добавляет запись в журнал.
body: "idTransfer", "idAccepted", "orders"
