API Справочник

Адрес запроса Описание
http://192.168.2.10:3131/orders GET Запрос: Возвращает массив заказов [orders], общее колличество заказов count и колличество страниц pages.
Параметры: _sort, _order, _page, _limit
http://192.168.2.10:3131/orders/ID(number) GET Запрос: Возвращает заказ order, шапку header, тело заказа body и планы plans.
Параметры: Нет
http://192.168.2.10:3131/orders/sample/ID(number) GET Запрос: Возвращает изображение образца, которое находиться в папке заказов "Образец".
Параметры: Нет
