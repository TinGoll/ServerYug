Структура проекта:

app/pools - папка с функцииями для pool запросов к базе данных
app/query - папка для sql-запросов к базе данных, используются в pool
config/.firebirdDB - папка содержит настройки подключения к базе данных
config/app.js - файл с настройками API (сейчас содержит только порт подключения)
config/routes*.js - файлы содержат пути для доступа извне к API
public - статичные HTML файлы API
